Data Preparation¶

We separate features (X) and target variable (y). Target variable is 'quality'.

In [9]:
X_red = red.drop("quality", axis=1)
y_red = red["quality"]

X_white = white.drop("quality", axis=1)
y_white = white["quality"]

Train-Test Split¶

We split the dataset into training and testing data. 80% data is used for training and 20% for testing.

In [12]:
from sklearn.model_selection import train_test_split

Xr_train, Xr_test, yr_train, yr_test = train_test_split(X_red, y_red, test_size=0.2, random_state=42)

Xw_train, Xw_test, yw_train, yw_test = train_test_split(X_white, y_white, test_size=0.2, random_state=42)

Model Training¶

We use Random Forest Classifier to train the model. It is an ensemble learning method that gives good accuracy.

In [13]:
from sklearn.ensemble import RandomForestClassifier

model_red = RandomForestClassifier()
model_red.fit(Xr_train, yr_train)

model_white = RandomForestClassifier()
model_white.fit(Xw_train, yw_train)

Predictions¶

We use trained model to predict wine quality on test data.

yr_pred = model_red.predict(Xr_test)

Accuracy Score¶

Accuracy shows how many predictions are correct out of total predictions.

Final Conclusion

  • White dataset gives better accuracy due to larger data size
  • Random Forest model performs well
  • Some overlapping classes reduce prediction accuracy

🏁 Final Conclusion

This project focused on predicting wine quality using machine learning techniques on both red and white wine datasets.



🔍 Key Findings:

  • The Random Forest model performed well in predicting wine quality.
  • The White wine dataset showed slightly better accuracy compared to red wine due to larger data size.
  • Some quality classes (especially 5 and 6) were harder to distinguish, leading to minor prediction errors.

📊 Overall Insight:
The model successfully captured important patterns in the dataset and provided reliable predictions, although overlapping classes slightly affected accuracy.



🚀 Conclusion:
Machine learning models like Random Forest can effectively predict wine quality and can be further improved with advanced techniques and feature engineering.
